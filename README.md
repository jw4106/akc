##Stack

 - Laravel (Artisan)
 - Blade for Views
 - Ajax, Jquery 
 - HTML/CSS

##instructions

- run composer install 
- config/database.php -> change to proper db 
- create schema 'akc'
- php artisan migrate
- php artisan migrate:fresh --seed
- php artisan key:generate
- php artisan config:cache
- php artisan serve
